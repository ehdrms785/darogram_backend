# prismagram

## User Stories

- [x] Create account
- [x] Request Secret
      [ ] Confirm Secret (Login)
- [x] Log in
- [x] Like / Unlike a photo
- [x] Comment on a photo
- [x] Search by user
- [x] Follow / Unfollow User
- [x] See the full photo
- [x] Edit my profile
- [x] Upload a photo
- [x] Edit the photo (Delete)
- [x] See the feed
